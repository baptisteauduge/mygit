char * convert_work_file_to_str ( const work_file_t *  wf)

Convert a work_file to a string, the string is allocated and must be freed. The format is "name|hash|mode" where the separator is defined by SEPARATOR_WORK_FILE_STRING constant. If the hash is NULL, the string will contain "name|-|mode". (With a single dash).

Parameters
wfThe work_file to convert
Returns
char* The string converted or NULL if any error occurred
