char * convert_work_tree_to_str ( const work_tree_t *  wt)

Convert a work_tree to a string, the format is "name|hash|mode\nname|hash|mode" where the separator is defined by SEPARATOR_WORK_FILE_STRING constant. If the hash is NULL, the string will contain "name|-|mode". (With a single dash). The format is based on the work_file_to_str function.

Parameters
wtThe work_tree to convert.
Returns
char* The string containing the work_tree or NULL if an error occurred.
